This lab requires the following:
  • Webex Meetings user credentials: You will need username/password/site name (e.g. 'myorg' if your Webex domain is myorg.Webex.com) to make API calls. The site must be enabled for XML API access by a site admin.

    A free-to-access demo Webex instance is available via the DevNet Sandbox: Webex Web Conferencing Sandbox

  • A REST API test client: You will need a REST test client, we recommend Postman
